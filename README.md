# SQL-Data-Analysis-Chinook-Database
This project demonstrates my skills in SQL for data analysis. I performed an exploratory analysis on the Chinook database to answer key business questions regarding sales performance, revenue trends, and top-selling products.
​What I did:
​Performed data aggregation and complex JOINs to link customer, invoice, and track data.
​Analyzed monthly revenue trends and identified top-performing products.
​Documented the findings and SQL queries in a comprehensive report.
​Tools: SQL (SQLite), DB Browser for SQLite, and PDF Reporting
